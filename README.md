# GAME DEVELOPER |UNITY|

สวัสดีครับผม นายคณาธิป พรายแก้ว ชื่อเล่น น๊อต ผมเป็นนักพัฒนาเกมแอคชั่นที่ชอบแนวการต่อสู้การผจญภัยและ effect ที่อลังการ
